## 🛠️ Technical Highlights

* **Pure Vanilla Stack:** Built exclusively with **HTML5** and **CSS3** to demonstrate core layout mastery without external frameworks.
* **Responsive Architecture:** Implemented using CSS Flexbox and Grid, ensuring the interface adapts perfectly from 4K displays to small-screen mobile devices.
* **Modern UI Components:** Features a sticky navigation bar, a high-impact hero section, and a balanced multi-column photo gallery.
* **Accessibility & SEO:** Uses semantic HTML tags (`<nav>`, `<section>`, `<footer>`) to improve search engine indexing and screen-reader compatibility.
